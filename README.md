# ETL Pipeline to Snowflake Data Warehouse

## Overview
This project demonstrates a batch ETL pipeline built using Python and SQL to load sales data into a Snowflake data warehouse.

## Architecture
CSV → Python (Extract & Transform) → Data Quality Checks → Snowflake Tables → Analytics Queries

## Technologies Used
- Python
- Snowflake
- Pandas
- SQL

## Features
- Batch data ingestion
- STAR-style schema design
- Data quality validations
- Warehouse loading
- Analytics-ready data

## Use Case
Designed to showcase ETL and Data Engineering skills for modern data platforms.
